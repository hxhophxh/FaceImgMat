║          Python 3.12.7 安装说明                                ║
╚════════════════════════════════════════════════════════════════╝

📦 本目录包含：
   python-3.11.9-amd64.exe  (约 25MB)

🎯 安装方法：

方法1：自动安装（推荐）
   运行上级目录的「一键完整部署.bat」会自动安装

方法2：手动安装
   1. 双击 python-3.11.9-amd64.exe
   2. 勾选 "Add Python to PATH"
   3. 选择 "Install Now"
   4. 等待安装完成

✅ 验证安装：
   打开命令提示符，输入：python --version
   应该显示：Python 3.11.9

⚠️  注意事项：
   - 安装路径: C:\Program Files\Python311
   - 需要约 150MB 磁盘空间
   - 建议以管理员身份安装
